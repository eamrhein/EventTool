## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- npm

```sh
npm install npm@latest -g
```

### Installation

1. Clone the EventTool

```sh
git clone https://github.com/eamrhein/EventTool.git
```

2. Install NPM packages

```sh
npm install
```

or

```sh
yarn install
```
